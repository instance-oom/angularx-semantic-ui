Options

  • data:  Required. Type of Array, when the array is stored in the Object, you need to specify the textField
  • textField:  Specify which properties are used to display on the page for object, Default is text
  • idField:  Specify which properties are used to uniq the item for object, Default is id
  • placeHolder:  Optional.
  • multiple:  Optional. Type of boolean. Default is false
